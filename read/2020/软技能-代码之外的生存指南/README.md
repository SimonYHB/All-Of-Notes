- 软技能：代码之外的生存指南

  1. 职业
     1. 像经营企业一样思考自己，能提供什么服务

     2. 设定目标，将大目标拆分成小目标

     3. 学会与人打交道

        - 尊重他人
        - 不要批评，不要急着否定
        - 换位思考
        - 避免争吵

     4. 面试技巧

        - 提前选好心怡的公司

        - 长期准备，不要临近面试才抱佛脚

        - 建立自身品牌
        - 通过与相关人员建立亲密关系来获取面试机会

     5. 专业选择： 专业化比全面更重要

     6. 晋升的方式

        - 成为专业人士

        - 积极承担更多责任
        - 引人注目
        - 持续的自学
        - 成为问题的解决者

     7. 提前半年做好辞职的准备

  2. 自我营销（打造个人品牌）
     1. 创建博客（公众号）
        - 通过评论他人优秀内容来引流
        - 分享在社交网络
     2. 确定主要目标
        1. 前期不要以赚钱作为目标
        2. 树立好的形象，吸引更多的流量，让更多人认识你
        3. 给人们想要的东西，越简单越好
     3. 应用社交媒体
        1. 选择1~2个主要社交媒体
        2. 保持活跃度
     4. 演讲、报告和培训
     5. 写出

  3. 学习
     1. 如何自我教育
        1. 学习知识的最好方式就是立即将其用于实践

        2. 通过实践积累问题，再回到书本或资料中寻找答案
        3. 将学到的内容教给别人

     2. 十步学习法

        1. 了解全局： 初步概览，了解“是什么”
        2. 确定范围： 根据学习理由和时间确定学习范围，务必大小适当
        3. 定义目标： 目标要明确，例如“使用HTML5在网上创建一个自己的主页”
        4. 寻找资料： 收集与主题相关的信息
        5. 创建学习计划： 参考资料结果列出主题大纲，细分到学习顺序和模块
        6. 筛选资料：通过浏览筛选4中获取的资料
        7. 开始学习，浅尝辄止： 学习基础知识
        8. 动手操作，边玩边学： 通过实践来发现更多的问题
        9. 全面掌握，学以致用： 通过查询资料解决8中记录的问题，深入学习模块，并回到7中开始学习下一个模块
        10. 乐为人师，融会贯通： 将学会的知识分享给他人，在整理的过程中发现没有摸透的问题

     3. 寻找导师

     4. 传道授业

     5. 拥有学位

     6. 清楚自己的短板

  4. 生产力
     1. 学会专注，减少工作时的干扰项
     2. 做好周/月/季度计划，每天开始工作时先浏览本周需要做的时间
     3. 使用番茄工作法，并约定好每周需要完成的番茄个数
     4. 对重复性任务采用定额工作法
     5. 对自己负责，避免违反自己制定的规则
     6. 不是所有任务都可以多任务并行处理
     7. 避免浪费时间（看电视、刷抖音等）
     8. 培养好习惯
     9. 分解任务，并确定每个小任务都有一个明确的目标
     10. 努力工作，树立正确的心态
     11. 采取行动，遇到问题再改变，而不是一直犹豫

  5. 理财
     1. 明确资产与负债
     2. 薪资谈判
        1. 被推荐优于投简历面试
        2. 不要先出价
        3. 学习谈判技巧
     3. 指定退休计划
     4. 避免承担非投资类型的负债

  6. 健身

     - 健身的好处

       - 培养自信心

       - 提升大脑的力量

       - 健康长寿

     - 如何健身

       - 设定健身目标，并拆分成小目标落实到每周
       - 注意卡路里的摄入和成分组成，健康硬是
       - 寻找动力，不要过早奖励自己

  7. 精神
     1. 相信自己能做到
     2. 保持积极乐观的心态
     3. 构建一个理想的自我形象
     4. 学会拥抱失败、期待失败、接收失败，并准备直面失败。
