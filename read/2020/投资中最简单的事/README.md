- 投资中最简单的事
  - 投资理念
    - 以实业的眼光做投资
      1. 是不是一门好生意
         - 有门槛和先发优势，行业集中度持续提高的行业
      2. 现金流状态如何
         - 宁要有门槛的低增长也不要无门槛的高增长，门槛好把握，而成长是将来的，难预测
      3. 行业的竞争格局以及公司作为后来者是否具有优势
         - 寡头垄断行业的利润增长容易超出预期，而市场集中度低的竞争压力大，容易陷入恶性竞争和价格战中
      
    - 逆向投资（人弃我取）

      - 新型行业看需求成长，传统行业看供给关系（寡头、产能提升）

      - 高估值购买新兴行业是沉迷于“未得到”，低估值购买夕阳行业是沉迷于“已失去”，收益比最高的还是容易被低估的“正拥有”的产业

      - 逆向投资的关键

        1. 看估值是否够低、是否已经过度反映了可能的坏信息，估值高的股票本身估值和增长预期下调空间大，刚开始时不宜你想投资

        2. 看遭遇的问题是否是短期问题，是否是可解决的问题，例如零售股面临网购冲击，新建城市综合体导致旧有商圈优势丧失，租金成本等上升就不是短期能解决的

           > 2020年国家鼓励地摊经济，但从大趋势看，根本问题无法解决，股价纵使上升也只是一时的

        3. 看股价暴跌本身是否会导致公司的基本面进一步恶化，即索罗斯说的反身性。中国的银行业因为有政府的隐形担保，不存在这种反身性，因此可以逆向投资

      - 食品饮料更适合做逆向投资，历史上看，发生安全事故时是较好的买入点，特别是那些没有直接卷入安全事故或牵涉程度浅的行业龙头企业

      - 逆向投资时考虑的问题

        - 是否有替代品，有则谨慎
        - 是个股问题还是行业问题，行业问题则谨慎，个股则重点研究其竞争对手
        - 是主动违规还是被动中枪，主动则谨慎
        - 是否容易解决，难以解决则谨慎
        - 企业根基是否扎实，不扎实则谨慎

      - 警惕热门行业，市场大多数时候是对的，但有的时候也可以错得很离谱

        - 市盈率被低估的好公司

    - 便宜才是硬道理

      - 买便宜货不一定赚钱，但赚钱的概率大。买便宜货往往先被套，最终赚钱

    - 总结
      
      - 做投资要研究的就是这些不以人的意志为转移的规律，而不是整天去猜测市场的情绪变化。有时猜测别人的情绪变化能给我们带来收益，但那是不能够持久的。而经济规律、行业特质、商业模式是客观存在的，你只要研究透了，它在三五年内是不会有大的变化的，能为理解这些规律的投资人提供持续的竞争优势。 

  - 投资方法

    - 投资的三个基本问题

      1. 估值

         - 投资分析工具
           1. 波特五力分析（好行业）：公司对上下游的议价权、与竞争对手的比较优势、行业对潜在进入者的门槛
           2. 杜邦分析（好公司）：弄清公司过去５年究竟是靠什么模式赚钱的（高利润、高周转还是高杠杆），然后看公司战略规划、团队背景和管理执行力等是否与其商业模 式一致
           3. 估值分析（好价格）：通过同业横比和历史纵比，加上市值与未来成长空间比，在显著低估时买入

      2. 品质

         - 是不是一个好行业
           - 的售价是不是显著高于其他竞争 者，领先者有没有网络服务的优势，有没有规模效应，产品的销售半径是不是相对比较小（不用参与全球竞争），是不是有回头客，是不是低单价（下游对价格不敏感），是不是转换成本高，领先者是不是有先发优势，技术变化是不是没有那么快。对品质的评判有很多指标，核心是“这是不是一门好生意，有没有定价 权，是不是一门容易赚钱的生意”。

         - 差异化竞争（决定定价权）
           - 美誉度而非知名度
           - 用户黏度高
           - 单价不要太高
           - 转换成本高越高
           - 服务网络广
           - 先发优势

      3. 时机

         - 历史上股市见底信号

           1. 市场估值在历史低位；
           2. M1见底回升；
           3. 降存准或降息；
           4. 成交量极度萎缩；
           5. 社保汇金入市；
           6. 大股东和高管增持； 

           7. 机构大幅超配非周期类股票；
           8. 强周期股在跌时抗跌，涨时领涨；
           9. 机构仓位在历史低点；
           10. 新股停发或降印花税。 
    - 宁数月亮，不数星星
      - 月朗星稀 > 一超多强 > 两分天下/三足鼎立 > 百花齐放
      - 数月亮即是找到寡头，在合适的价格买入
    - 经验
      - 在百花齐放阶段，不必急着下注，可等着龙头群明显是再挑最强的买入
      - 对于散户来说，便宜的好公司是较好的选择，安全边际高
      - 在市场的底部区域，便宜的好公司到处皆是，只是此时往往有钱的没胆，有胆的没钱

  - 投资风险

    - 价值陷阱与成长陷阱
      - 容易陷入价值陷阱的股票类型
        - 被技术进步淘汰的
        - 赢家通吃行业里的小公司
        - 分散的、重资产的夕阳行业
        - 景气顶点的周期股
      - 成长陷阱
        - 估值过高
        - 技术路径踏空
        - 无利润增长
        - 成长性破产
        - 盲目多元化
        - 在门槛低的行业树大招风
        - 新产品风险
        - 寄生式增长（例如为苹果提供零件）
        - 强弩之末
        - 会计造假
      - 成长的持续性
        - 宁要有门槛的地增长，不要没门槛的高增长，因为前者可持续，后者难维系
      - 估值
        - A股中的股票估值高低与公司品质的好坏公司不大
    - 真假风险与安全边际
