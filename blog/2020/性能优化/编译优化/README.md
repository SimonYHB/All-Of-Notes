### 编译流程

编译器分为前端和后端

Xcode3之前前端编译器为 GCC，后面自研 Clang+LLVM 和 Swift+LLVM

```
1. 源代码（source code） 
2. 预处理器（preprocessor） 
3. 编译器（compiler） 
4. 汇编程序（assembler） 
5. 目标代码（object code） 
6. 链接器（Linker） 
7. 可执行文件（executables）
```
